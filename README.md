# Aphid_Vectored_Fungal_Epidemics

This is an R Markdown file showing how all figures in Simon et al (202x) "Fungal secondary metabolism determines aphid vector competence" are created

There are no particular requirements in terms of operating system or version of R, so long as the following libraries can be used
- readxl
- deSolve
- fields
- sfsmisc
- extrafont
- RColorBrewer

A data file that is required Within Head Severity Data.xlsx is also included as part of the repository.
