Fungal secondary metabolism determines aphid vector competence

This R markdown file contains R code used in the epidemiological modelling component of Simon et al. (2026) “Fungal secondary metabolism determines aphid vector competence”.

All figures in the paper can be produced using this file.

Parameterisation of epidemiological model

Report the default parameterisation

# parameters
defParamNIV <- list(K = 300,     # carrying capacity for spikes per unit area
                    mu = 0.1,    # rate of production of spikes
                    r = 0.15,    # aerial infection rate
                    Gamma = 1,   # time an aphid feeds for
                    alpha = 0.5, # probability aphid picks up fungus per visit
                    beta = 0.5,  # probability aphid transmits fungus per visit
                    v = 1.2,     # vector preference (fungus-free vectors to infected plants)
                    a = 0.5,     # aphid birth rate  
                    eta = 0.2,   # rate of loss of infectivity 
                    c = 0.2,     # death rate of aphids
                    M = 800,     # "carrying capacity" of aphids per unit area 
                    delta = 1.5, # vector preference (fungus-carrying vectors to healthy plants)
                    # Severity parameters calculated by model fitting below
                    omega = 0.25,      # rate at which severity increases on a spike
                    zetaZero = 0.034)  # initial proportion of spikelets infected

defParamDON <- defParamNIV
# Three changes for DON relative to NIV:
#     - 1% as likely to pick up fungus from infected head as NIV
#     - 50% as likely to choose infected head as NIV 
#     - 50% more transmissible via aerial spread
defParamDON$alpha <- 0.01 * defParamNIV$alpha
defParamDON$v <- 0.5 * defParamNIV$v
defParamDON$r <- 1.5 * defParamNIV$r
defParamDON$delta <- 1.0

The results of this code, i.e., the best fitting parameters

  • \(\zeta_0 = 0.034\) (severity at time of first infection)
  • \(\omega = 0.25\) day\(^{-1}\) (rate of increase of severity).

are used in generating the figures above.
